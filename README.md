# mod_helloworld
This is a Node.js module to print Hello World in the console

# Installation
```sh
npm install mod_helloworld --save
```
## Usage

### Javascript

```javascript
var helloworld = require('mod_helloworld');
helloworld.HelloWorld();
```
```sh
Output should be 'Hello World !!!' in the console
```
